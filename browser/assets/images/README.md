# Images Directory

## Required Images

### OG Image (Open Graph)
- **File**: `og-image.jpg`
- **Size**: 1200x630 pixels (recommended)
- **Format**: JPG or PNG
- **Purpose**: Social media preview image (Facebook, Twitter, LinkedIn)
- **Content**: Should include your name, title, and portfolio branding

### Logo
- **File**: `logo.png`
- **Size**: 512x512 pixels (square)
- **Format**: PNG with transparency
- **Purpose**: Used in structured data and branding

## Creating OG Image

You can create the OG image using:
- Canva (https://www.canva.com)
- Figma
- Photoshop
- Any design tool

**Recommended OG Image Content:**
- Your name: "Yuvraj Bagale"
- Your title: "Frontend Developer"
- Portfolio tagline or key skills
- Professional photo (optional)
- Brand colors matching your portfolio

**Example Dimensions:**
- Width: 1200px
- Height: 630px
- Safe area: Keep important content within 1200x630px
